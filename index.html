STEP 1: BROADCASTING SOURCE

You need a server to host your audio stream (Icecast/Shoutcast).

* SOFTWARE FOR LIVE: Use Mixxx (Free) or VirtualDJ. Connect them to your server URL via Icecast protocol.

STEP 2: TELEGRAM INTEGRATION

How to launch this "Mini App" in your bot.

  1. Host this HTML file on a static site provider with HTTPS (required).
    SUGGESTIONS: GitHub Pages, Vercel, or Netlify.
  2. Open Telegram and search for @BotFather.
  3. Send command /newapp.
  4. Select your bot.
  5. Enter Title (e.g., Club Metaghetto) and Description.
  6. Upload a 640x360 photo.
  7. CRITICAL: Paste your https://your-site.vercel.app URL when asked for "Web App URL".
  8. BotFather will give you a short link (t.me/yourbot/appname).
